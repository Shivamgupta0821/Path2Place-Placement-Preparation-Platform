Check if an Array is Sorted

Given an array of integers, determine whether the array is sorted in non-decreasing order.

Example 1
Input: [1, 2, 3, 4, 5]
Output: true
Example 2
Input: [1, 3, 2, 4]
Output: false
Constraints
1 ≤ n ≤ 10⁵
-10⁹ ≤ arr[i] ≤ 10⁹
Hint

Compare each element with the previous one. If you ever find a smaller value after a larger one, the array is not sorted.
